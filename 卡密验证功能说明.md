### 2. 设备唯一标识（UDID）

#### 生成算法
```javascript
function generateUDID() {
  // 1. 创建Canvas指纹
  const canvas = document.createElement('canvas');
  const ctx = canvas.getContext('2d');
  ctx.textBaseline = 'top';
  ctx.font = '14px Arial';
  ctx.fillText('Browser fingerprint', 2, 2);
  const fingerprint = canvas.toDataURL();
  
  // 2. 收集设备信息
  const userAgent = navigator.userAgent;
  const screenInfo = `${screen.width}x${screen.height}x${screen.colorDepth}`;
  const timezone = Intl.DateTimeFormat().resolvedOptions().timeZone;
  
  // 3. 组合唯一字符串
  const uniqueString = fingerprint + userAgent + screenInfo + timezone;
  
  // 4. 生成哈希值
  let hash = 0;
  for (let i = 0; i < uniqueString.length; i++) {
    const char = uniqueString.charCodeAt(i);
    hash = ((hash << 5) - hash) + char;
    hash = hash & hash; // 转换为32位整数
  }
  
  // 5. 生成最终UDID
  return 'DAS' + Math.abs(hash).toString(36).toUpperCase().substring(0, 5);
}
```

#### 设备信息组成
- **Canvas指纹**：浏览器渲染引擎特征
- **用户代理**：浏览器版本和系统信息
- **屏幕信息**：分辨率、色深等显示参数
- **时区信息**：用户所在时区

### 3. 认证状态管理

#### 状态结构
```javascript
const authState = {
  isAuthenticated: false,    // 是否已认证
  token: null,              // 认证令牌
  cardNo: null,             // 卡密号码
  vipExpTime: null,         // 到期时间戳
  udid: null                // 设备唯一标识
};
```

#### 状态保存
```javascript
async function saveAuthState() {
  try {
    await chrome.storage.local.set({
      authState: {
        ...authState,
        timestamp: Date.now()  // 保存时间戳
      }
    });
  } catch (error) {
    console.error('保存认证状态失败:', error);
  }
}
```

#### 状态恢复
```javascript
async function restoreAuthState() {
  try {
    const data = await chrome.storage.local.get('authState');
    if (data.authState) {
      // 检查24小时内是否有效
      const now = Date.now();
      const savedTime = data.authState.timestamp || 0;
      const hoursDiff = (now - savedTime) / (1000 * 60 * 60);
      
      if (hoursDiff < 24) {
        authState = { ...data.authState };
        delete authState.timestamp;
        
        // 验证token是否还有效
        if (authState.token && authState.vipExpTime) {
          const currentTime = Math.floor(Date.now() / 1000);
          if (currentTime < authState.vipExpTime) {
            return true; // 认证有效
          }
        }
      }
    }
  } catch (error) {
    console.error('恢复认证状态失败:', error);
  }
  return false;
}
```

### 4. 安全检查机制

#### 定期检查
```javascript
function startAuthCheck() {
  // 每5分钟检查一次认证状态
  setInterval(async () => {
    if (authState.isAuthenticated && authState.vipExpTime) {
      const currentTime = Math.floor(Date.now() / 1000);
      if (currentTime >= authState.vipExpTime) {
        // 卡密已到期
        addLog('卡密已到期，请重新验证', 'error');
        authState.isAuthenticated = false;
        await chrome.storage.local.remove('authState');
        switchToAuthContent();
        showAuthStatus('卡密已到期，请重新验证', 'error');
      }
    }
  }, 5 * 60 * 1000); // 5分钟检查一次
}
```

#### 版本检查

```javascript
// 检查版本是否匹配

  throw new Error(`版本不匹配，当前版本：${AUTH_CONFIG.VERSION}，服务器版本：${data.info.version}，请下载最新版本`);

```

#### 到期检查
```javascript
// 检查卡密是否到期
const currentTime = Math.floor(Date.now() / 1000);

```

## 🛡️ 安全特性

### 1. 设备绑定
- **唯一标识**：每个浏览器生成唯一的UDID
- **防多设备**：同一卡密无法在多个设备使用
- **持久绑定**：UDID与卡密绑定，更换设备需要重新验证

### 2. 自动检查
- **定期验证**：每5分钟检查一次认证状态
- **到期提醒**：卡密到期自动提示并阻止使用
- **状态同步**：实时同步服务器端卡密状态

### 3. 状态保存
- **本地存储**：认证状态保存在浏览器本地存储
- **时间限制**：24小时内无需重复验证
- **自动清理**：过期状态自动清理

### 4. 版本控制
- **版本检测**：自动检测插件版本
- **升级提醒**：版本不匹配时提示升级
- **向后兼容**：支持版本升级检测
