# 小红书自动发布插件 v3.0

一个功能强大的Chrome浏览器插件，支持自动发布小红书笔记，现已集成卡密验证系统。

## 🔐 卡密验证功能

### 安全特性
- **设备绑定**：卡密与浏览器唯一标识绑定，防止多设备使用
- **自动检查**：插件会定期检查卡密状态，到期自动提示
- **状态保存**：验证成功后24小时内无需重复验证
- **版本控制**：支持版本升级检测，确保使用最新版本

### 验证流程
1. 打开插件后，首先会显示卡密验证界面
2. 输入您的卡密（请联系客服购买）
3. 点击"登录验证"按钮
4. 验证成功后自动进入主功能界面

## ✨ 主要功能

### 📁 文件导入
- 支持CSV、Excel、TXT多种格式
- 自动解析笔记标题、正文、标签、商品ID
- 支持批量导入图片链接

### 🖼️ 图片处理
- 支持CSV导入图片链接自动下载
- 支持手动上传本地图片
- 支持拖拽调整图片顺序
- 支持删除和重新排序

### ⏰ 智能发布
- 支持固定间隔和随机间隔发布
- 自动填写笔记内容
- 自动添加标签
- 自动关联商品（如有商品ID）
- 实时发布状态监控

### 📊 日志监控
- 实时显示发布进度
- 详细的错误信息提示
- 历史日志记录保存
- 倒计时显示

## 🚀 快速开始

### 1. 安装插件
1. 下载插件文件
2. 打开Chrome浏览器，进入扩展程序页面
3. 开启开发者模式
4. 点击"加载已解压的扩展程序"
5. 选择插件文件夹

### 2. 卡密验证
1. 点击浏览器工具栏中的插件图标
2. 在验证界面输入您的卡密
3. 点击"登录验证"
4. 验证成功后进入主功能界面

### 3. 导入笔记
1. 准备CSV格式的笔记文件
2. 点击"选择文档文件"
3. 选择您的文件
4. 预览和编辑笔记内容

### 4. 添加图片
1. 在笔记面板中点击"添加图片"
2. 选择本地图片文件
3. 拖拽调整图片顺序
4. 或通过CSV文件导入图片链接

### 5. 设置发布间隔
1. 选择固定间隔或随机间隔
2. 设置合适的发布间隔时间
3. 建议设置5-15分钟间隔

### 6. 开始发布
1. 确保已登录小红书创作者平台
2. 点击"开始运行"
3. 插件自动执行发布流程
4. 实时监控发布状态

## 📋 CSV文件格式

```csv
标题,正文,标签,商品ID,主图链接
笔记标题1,这是第一篇笔记的正文内容,#标签1 #标签2,PROD001,http://example.com/image1.jpg
笔记标题2,这是第二篇笔记的正文内容,#标签3 #标签4,PROD002,http://example.com/image2.jpg
```

### 字段说明
- **标题**：笔记标题（必填）
- **正文**：笔记正文内容（必填）
- **标签**：以#开头的标签，多个标签用空格分隔
- **商品ID**：关联商品的ID（可选）
- **主图链接**：图片链接，支持多个链接用换行符分隔

## ⚠️ 注意事项

### 使用前准备
- 确保已登录小红书创作者平台
- 确保网络连接稳定
- 确保卡密有效且未到期
- 建议在发布前测试单篇笔记

### 安全提醒
- 不要设置过短的发布间隔
- 避免同时发布大量笔记
- 定期检查发布效果
- 如遇异常及时停止发布

### 最佳实践
- 建议每天发布不超过10篇笔记
- 设置合理的发布间隔（5-15分钟）
- 确保笔记内容原创且高质量
- 定期更新卡密，确保服务正常

## 🆘 常见问题

### 卡密相关问题
**Q: 卡密验证失败怎么办？**
A: 请检查卡密是否正确，网络是否正常，或联系客服确认卡密状态。

**Q: 卡密到期了怎么办？**
A: 需要重新购买卡密，插件会自动提示并引导重新验证。

**Q: 可以在多个设备使用吗？**
A: 不可以，卡密绑定浏览器唯一标识，防止多设备使用。

### 发布相关问题
**Q: 发布失败怎么办？**
A: 检查网络连接，确保已登录小红书，查看日志中的具体错误信息。

**Q: 图片上传失败怎么办？**
A: 检查图片格式和大小，确保图片链接可访问。

**Q: 如何停止发布？**
A: 点击"终止程序"按钮，插件会安全停止当前发布任务。
