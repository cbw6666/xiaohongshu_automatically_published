# 小红书自动发布插件 - 多图片链接功能优化

## 功能概述

本次优化主要解决了CSV导入时主图链接字段包含多个图片链接的问题。现在插件能够正确识别和导入以http或https开头的多个图片链接，并在笔记预览区域显示对应的图片。

## 主要修改

### 1. 图片链接解析逻辑优化

**文件**: `popup.js`
**位置**: `parseCsvNotes` 函数中的图片链接处理部分

**修改前**:
```javascript
// 主图链接支持多张，安全处理
let imageLinks = [];
if (idx.image >= 0 && idx.image < row.length && row[idx.image]) {
  imageLinks = row[idx.image].split(/[,，]/).map(s => s.trim()).filter(Boolean);
}
```

**修改后**:
```javascript
// 主图链接支持多张，安全处理 - 支持换行符分隔的多个链接
let imageLinks = [];
if (idx.image >= 0 && idx.image < row.length && row[idx.image]) {
  const imageField = row[idx.image];
  addLog(`第${i+1}行主图字段原始内容: ${imageField}`, 'info');
  
  // 先按换行符分割，再按逗号分割，然后过滤出http/https链接
  const allLinks = imageField
    .split(/[\n\r,，]/) // 支持换行符、回车符、逗号、中文逗号分隔
    .map(s => s.trim())
    .filter(s => s && (s.startsWith('http://') || s.startsWith('https://')))
    .map(s => s.replace(/[""]/g, '')); // 移除可能的引号
  imageLinks = allLinks;
  
  addLog(`第${i+1}行解析出 ${imageLinks.length} 个有效图片链接: ${imageLinks.join(', ')}`, 'success');
}
```

### 2. 支持的链接格式

现在插件支持以下格式的图片链接：

1. **换行符分隔**:
   ```
   https://example.com/image1.jpg
   https://example.com/image2.jpg
   https://example.com/image3.jpg
   ```

2. **逗号分隔**:
   ```
   https://example.com/image1.jpg,https://example.com/image2.jpg,https://example.com/image3.jpg
   ```

3. **混合分隔符**:
   ```
   https://example.com/image1.jpg
   https://example.com/image2.jpg,https://example.com/image3.jpg
   ```

4. **包含无效链接**:
   ```
   https://example.com/image1.jpg
   无效链接
   https://example.com/image2.jpg
   ftp://example.com/image3.jpg
   ```

### 3. 链接验证规则

- 只接受以 `http://` 或 `https://` 开头的链接
- 自动过滤空字符串和无效链接
- 自动移除链接两端的引号
- 支持中英文逗号分隔

### 4. 日志增强

添加了详细的解析日志，包括：
- 每行主图字段的原始内容
- 解析出的有效图片链接数量
- 具体的图片链接列表

## 使用示例

### CSV文件格式示例

```csv
主图链接,标题,正文,标签,商品ID
https://example.com/image1.jpg
https://example.com/image2.jpg
https://example.com/image3.jpg,测试笔记1,这是正文内容,#测试标签1 #测试标签2,12345
https://example.com/image4.jpg,https://example.com/image5.jpg,测试笔记2,这是正文内容,#测试标签3,67890
```

### 导入流程

1. 点击"选择文件"按钮
2. 选择包含多图片链接的CSV文件
3. 插件自动解析并下载图片
4. 在笔记预览区域显示图片
5. 可以拖拽调整图片顺序
6. 可以删除不需要的图片

## 注意事项

1. 图片链接必须是可访问的URL
2. 建议使用https链接以确保安全性
3. 大量图片下载可能需要一些时间
4. 网络问题可能导致部分图片下载失败

## 技术实现

- 使用正则表达式 `/[\n\r,，]/` 分割链接
- 使用 `startsWith()` 方法验证链接格式
- 使用 `fetch()` API 下载图片
- 使用 `FileReader` 转换为base64格式
- 使用 `localStorage` 保存图片数据
